void buttons_init ( void )

Inicializa os pinos GPIO para os botões. Configura os pinos como entrada com pull-up interno.

Inicializa os pinos GPIO para os botões. Configura os pinos como entrada com pull-up interno.

Esta função configura os pinos GPIO para os botões A e B como entradas com resistores pull-up internos habilitados. Isso permite detectar quando os botões são pressionados (conectados ao GND).

◆ buttons_read()

void buttons_read ( ButtonStates_t * states)

Lê o estado atual dos botões.

Parâmetros
statesPonteiro para a estrutura ButtonStates_t onde o estado será armazenado.

Esta função lê o estado atual dos pinos GPIO conectados aos botões A e B e armazena os resultados na estrutura ButtonStates_t fornecida. Com resistores pull-up, quando o botão é pressionado, o pino vai para estado LOW.

Parâmetros
statesPonteiro para a estrutura ButtonStates_t onde o estado será armazenado
Observação
O estado de um botão será considerado como pressionado (true) quando o pino estiver em LOW
