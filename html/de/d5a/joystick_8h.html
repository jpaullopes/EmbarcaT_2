Descrição detalhada

Interface para controle e leitura do joystick.

Este arquivo contém as definições para interagir com o joystick, incluindo inicialização e leitura das posições X/Y e estado do botão.

◆ joystick_init()

void joystick_init ( void )

Inicializa o joystick.

Configura os pinos GPIO e inicializa o ADC para leitura do joystick. Deve ser chamada antes de qualquer outra função do joystick.

Configura o ADC e os pinos GPIO necessários para o joystick:

  • Inicializa o ADC
  • Configura os pinos analógicos X e Y
  • Configura o pino do botão com pull-up interno

◆ read_joystick()

void read_joystick ( Joystick * joystick)

Lê os valores do joystick.

Captura os valores atuais de posição X, Y e o estado do botão. Os valores X e Y são normalizados para o intervalo de 0-100.

Parâmetros
joystickPonteiro para a estrutura onde serão armazenados os valores lidos

Realiza a leitura dos valores de X e Y do joystick através do ADC e do estado do botão através da GPIO. Os valores são normalizados para uma escala de 0-100 para facilitar o processamento.
